# 系统概览

EdgeEMS 是面向多站点、多设备的边缘能源管理系统。系统通过统一的设备接入、点位配置与告警策略，完成监控、控制与统计分析的闭环管理。

## 适用角色

- **Viewer**：只读查看数据、报表与告警。
- **Operator**：可执行控制操作与日常维护。
- **Admin**：配置设备、点位、用户权限与系统参数。

## 入口与导航

- 左侧导航为主功能入口，按“监控 / 告警 / 控制 / 统计 / 系统管理”组织。
- 顶部状态栏展示当前站点、实时告警数与账号信息。
- 右上角支持快速切换站点或项目群。

## 数据范围

- 监控：实时功率、能量、设备状态与运行趋势。
- 告警：阈值告警、通讯告警与设备异常。
- 统计：日/月/年报表与异常对比分析。
