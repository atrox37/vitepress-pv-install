# 首次使用流程

本章节帮助你在 30 分钟内完成首次接入与基本配置。

## 1. 创建站点

- 进入“系统管理 / 系统配置”
- 填写站点名称、时区、容量与负责人

## 2. 添加设备

- 进入“系统管理 / 设备配置”
- 选择设备类型（电池、PCS、光伏等）
- 填写设备编码与通讯参数

## 3. 配置通道与点位

- 进入“系统管理 / 通道与点位”
- 选择协议模板或手动配置
- 绑定设备与点位集合

## 4. 监控与告警

- 在“设备监控”中确认实时数据正常
- 在“告警中心”中设置阈值与通知策略

## 5. 发布控制策略

- 进入“控制管理”
- 配置策略与触发条件
- 进行试运行并发布

::: tip
若首次接入时数据为空，请先确认设备通信与点位绑定是否完成。
:::
