1. When installing photovoltaic modules, use module installation positioning blocks to center the photovoltaic modules (when using module installation positioning blocks, place them uniformly on one side)

<div class="mc-img-row">
  <img src="/images/step/step5/（3）/①.1.png" alt="Step 5-③-①.1" />
  <!-- <img src="/images/step/step5/（3）/①.2.png" alt="Step 5-③-①.2" /> -->
</div>

2. Place one side of the photovoltaic panel tightly against one side of the module installation positioning block on the component beam frame (place the photovoltaic panel positive wire box on the lower shaft rod combination A side, negative on the upper shaft rod combination side, the positive and negative orientation of 16 photovoltaic panels in two groups should be placed uniformly according to the first panel of the first group)

<div class="mc-img-row">
  <img src="/images/step/step5/（3）/②.1.png" alt="Step 5-③-②.1" />
</div>

3. Then use 4 clamping blocks to press the photovoltaic module, the clamping blocks should be tight against the module aluminum frame edge

<div class="mc-img-row">
  <img src="/images/step/step5/（3）/③.1.png" alt="Step 5-③-③.1" />
  <!-- <img src="/images/step/step5/（3）/③.2.png" alt="Step 5-③-③.2" /> -->
</div>

4. Use a torque wrench with 20N.m force to connect the U-bolts and clamping blocks to press the photovoltaic panel (all U-bolts in the lower shaft rod direction should be tightened tight against the node reinforcement piece edge)

Adjust the torque wrench torque value to 20N.m and use a 13mm socket to tighten.

  <img src="/images/step/step5/（3）/④.2.png" alt="Step 5-③-④.2" />
</div>
